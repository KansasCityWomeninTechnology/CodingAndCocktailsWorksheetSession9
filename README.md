### Helpful Keyboard Shortcuts
|Command|Mac Key Combination|Windows Key Combination|
|--|--|--|
|Cut|`cmd` + `x`|`ctrl` + `x`|
|Copy|`cmd` + `c`|`ctrl` + `c`|
|Paste|`cmd` + `v`|`ctrl` + `v`|
|Undo|`cmd` + `z`|`ctrl` + `z`|
|Redo|`cmd` + `y`|`ctrl` + `y`|
|Save|`cmd` + `s`|`ctrl` + `s`|
|Select all|`cmd` + `a`|`ctrl` + `a`|
|Find|`cmd` + `f`|`ctrl` + `f`|
